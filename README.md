## Installation

 ### Run Locally

- Download Zip
- Open the Project using VS Code(recommended)
- Install required node modules

```bash
  npm i express body-parser
```

- Run the server

```bash
  nodemon index.js
```



- Enjoy!

```bash
  nodemon index.js
```
